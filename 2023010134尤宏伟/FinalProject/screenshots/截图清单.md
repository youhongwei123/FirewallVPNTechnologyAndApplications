## 第一部分：网络规划与基础搭建（4张）

- **操作**: 运行 `setup.sh` 后执行

### 截图 04: 基础连通性测试
- **文件名**: `04-base-ping.png`

## 第二部分：防火墙策略（5张）

### 截图 05: iptables FORWARD 链规则
- **文件名**: `05-iptables-forward.png`
- **操作**: 运行 `firewall.sh` 后的输出
- **命令**: `ip netns exec fw iptables -L FORWARD -n -v --line-numbers`
- **预期内容**: 显示所有 FORWARD 链规则，包括 ACCEPT/REJECT/LOG 规则

### 截图 06: iptables NAT 表规则
- **文件名**: `06-iptables-nat.png`
- **操作**: 运行 `firewall.sh` 后的输出
- **命令**: `ip netns exec fw iptables -t nat -L -n -v --line-numbers`
- **预期内容**: 显示 MASQUERADE(SNAT) 和 DNAT 规则

### 截图 07: office → dmz:8080 访问成功
- **文件名**: `07-office-dmz-pass.png`
- **操作**: 先在 dmz 启动 Web 服务，再从 office 访问
- **命令**:
  - `ip netns exec dmz python3 -m http.server 8080 &`
  - `ip netns exec office curl http://10.40.0.2:8080/`
- **预期内容**: curl 返回 HTTP 200，显示目录列表页面

### 截图 08: office → dmz:22 访问被拒绝
- **文件名**: `08-office-dmz-ssh-deny.png`
- **操作**: 从 office 访问 dmz 的 22 端口
- **命令**: `ip netns exec office curl --max-time 3 http://10.40.0.2:22/`
- **预期内容**: 连接被拒绝 (Connection refused)

### 截图 09: guest 隔离测试
- **文件名**: `09-guest-isolation.png`
- **操作**: 从 guest 尝试访问 office 和 dmz
- **命令**:
  - `ip netns exec guest curl --max-time 2 http://10.20.0.2:8000/`
  - `ip netns exec guest curl --max-time 2 http://10.40.0.2:8080/`
  - `ip netns exec guest curl --max-time 2 http://203.0.113.10/`
- **预期内容**: office 和 dmz 访问被拒绝，internet 访问成功（NAT）

### 截图 10: DNAT 外网访问测试
- **文件名**: `10-dnat-test.png`
- **操作**: 从 internet 通过 DNAT 访问 dmz Web 服务
- **命令**: `ip netns exec internet curl http://203.0.113.1:8080/`
- **预期内容**: curl 返回 HTTP 200，证明 DNAT 端口转发成功

## 截图要求

1. **格式**: PNG 格式
2. **内容**: 必须包含完整的命令和输出结果
3. **清晰度**: 文字清晰可读
4. **命名**: 按序号命名，如 `01-ip-netns-list.png`
5. **存放**: 统一放在 `screenshots/` 目录下
6. **数量**: 至少 20 张，覆盖所有5个部分

## 快速截图指南

```bash
# 方式1: 使用 script 命令录制终端（推荐）
script -c "sudo bash test.sh" screenshots/test-output.txt
# 然后截取终端输出

# 方式2: 使用 gnome-screenshot
gnome-screenshot -d 2 -f screenshots/01-ip-netns-list.png

# 方式3: 使用 import 命令（ImageMagick）
import screenshots/01-ip-netns-list.png

# 方式4: 手动截图（Print Screen键）
# 在终端中执行命令后按 Print Screen 截图
```
